Portfolio Argentina Programa

Aplicación web full stack, que muestra mis datos personales, estudios cursados, experiencia laboral, conocimiento de las tecnologías. Esta aplicación es de arquitectura distribuida y contiene un diseño de interfaz de usuario (front end) que muestra la información, una base de datos que almacena los datos antes mencionados y cuenta con las APIs necesarias para proveer a través de internet en la nube la información (back end).
